Bootstrap 网格系统

Bootstrap 包含了一个响应式的、移动设备优先的、不固定的网格系统，可以 随着设备或视口大小的增加而适当地扩展到 12 列。它包含了用于简单的布局选项的预定 义类，也包含了用于生成更多语义布局的功能强大的混合类。
